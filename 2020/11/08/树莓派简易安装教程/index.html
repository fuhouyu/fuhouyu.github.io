# 开启 vnc 功能

选择Interface Options

选择VNC

确认后如下图，即开启成功：

![image-20201108122543166](/Users/fuhouyu/Library/Application Support/typora-user-images/image-20201108122543166.png)

确定回到主页面，Finish退出

使用VNC软件连接，输入ip和用户名和密码，即可进入桌面

如果在进入桌面时出现错误提示，可能是分辨率导致，需要修改分辨率：

选择Display Options

选择Resolution

![image-20201108123455678](/Users/fuhouyu/Library/Application Support/typora-user-images/image-20201108123455678.png)
