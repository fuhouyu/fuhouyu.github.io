# 简介

Minio 是一个基于 Apache License v2.0 开源协议的对象存储服务。它兼容亚马逊 S3 云存储服务接口，非常适合于存储大容量非结构化的数据，例如图片、视频、日志文件、备份数据和容器 / 虚拟机镜像等，而一个对象文件可以是任意大小，从几 kb 到最大 5T 不等。

MinIO 是一个非常轻量的服务，可以很简单的和其他应用的结合，类似 NodeJS, Redis 或者 MySQL。

# Go 语言安装

Minio的安装方式有多种，这里使用GO语言对Minio进行安装

  1. 安装 go 语言：Go 语言社区

    # 下载源码
wget https://golang.google.cn/dl/go1.15.5.linux-arm64.tar.gz
# 解压并拷贝到/usr/local下
tar -C /usr/local -zxf go1.15.5.linux-arm64.tar.gz\

  2. 修改环境变量

    vim /etc/profile

  3. 增加 Go 的环境变量

    export PATH=$PATH:/usr/local/go/bin
# 刷新变量
source /etc/profile

    输入 go version 验证

  4. 设置代理，否则一些 GO 的模块只能科学下载了

    go env -w GOPROXY=https://goproxy.cn,direct
export GOPROXY=https://goproxy.io/

# 安装 Minio

输入如下代码进行安装：

GO111MODULE=on go get github.com/minio/minio

等待下载完成，视对应的网络环境而定

进入文件夹查看 cd /usr/local/go/bin 是否存在 minio 的命令，如果不存在可使用全局查找， find /* -name minio 再将命令移动到当 go 的 bin 目录下

运行 minio

# /data 是文件上传的路径，可以设置为其他的地址
minio server /data

这上面显示的就是外部可以进行访问的地址，默认的账号密码为： minioadmin

使用自定义的账号密码

export MINIO_ACCESS_KEY=admin
export MINIO_SECRET_KEY=password
minio server /data

# 添加 Minio 的自启动

# 打开/etc/rc.local目录
vim /etc/rc.local
# 添加以下代码
export MINIO_ACCESS_KEY=帐号
export MINIO_SECRET_KEY=密码
nohup ./usr/local/go/bin/minio server /data > /ddhome/minio.log 2>&1 &

如图:

这时候 reboot 重启，就可以进行自启动了，如果不能执行自启动的操作，查看 rc.local 是否有可执行权限。

# 添加权限
chmod +x /etc/rc.local
